# PersonalWeb (Next.js)

本项目是可直接部署到 Vercel 的 Next.js（App Router）项目。

## 本地开发

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`。

## 部署到 Vercel

- 直接把当前仓库导入 Vercel（Framework 选择 Next.js，通常会自动识别）
- Build Command: `npm run build`
- Output: 默认（Next.js）
